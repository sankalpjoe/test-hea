# Violence Detection App

This application monitors video streams and detects violent or harmful activities in real-time using OpenAI's CLIP, OpenCV, and PyTorch.

## Features

- Real-time video stream analysis (webcam or video file)
- Violence detection using zero-shot CLIP model
- Basic headcount and crowd detection
- Configurable settings via settings.yaml

## Requirements

- Python 3.7+
- OpenCV
- PyYAML
- NumPy
- PyTorch

Install dependencies:

```bash
pip install -r requirements.txt
```

## Running the App

```bash
python app.py
```
